# OptionGroup

Single or multi-select option group with vertical/horizontal layout using [OptionCards](/components/option-card).

- **Tier 2** — requires `react-native-reanimated`
- **Registry deps:** [OptionCard](/components/option-card)

```bash [Terminal]
npx popapp add option-group
```

## Single select

<PreviewImage src="/images/option-group-single-select.png" alt="Single select" />

```tsx
import { OptionGroup } from "@/components/ui/option-group";

<OptionGroup
  mode="single"
  value={size}
  onChange={setSize}
  options={[
    { value: "small", label: "Small", description: "640 x 480" },
    { value: "medium", label: "Medium", description: "1280 x 720" },
    { value: "large", label: "Large", description: "1920 x 1080" },
  ]}
/>
```

## Multi select


## Horizontal layout

Use `layout="horizontal"` for side-by-side selection (great for binary choices):

```tsx
<OptionGroup
  mode="single"
  layout="horizontal"
  value={preference}
  onChange={setPreference}
  options={[
    { value: "window", label: "Window", icon: <IconSymbol name="sun.max.fill" size={24} /> },
    { value: "aisle", label: "Aisle", icon: <IconSymbol name="figure.walk" size={24} /> },
  ]}
/>
```

Horizontal layout automatically uses `align="center"` on each card, stacking icon and label vertically.

## Types

```tsx
interface OptionGroupOption {
  value: string;
  label: string;
  description?: string;
  icon?: ReactNode;
}
```

## Props

The component uses a discriminated union based on `mode`:

### Single select

| Prop | Type | Description |
|------|------|-------------|
| `mode` | `"single"` | Single selection mode |
| `value` | `string \| null` | Selected value |
| `onChange` | `(value: string) => void` | Change handler |
| `options` | `OptionGroupOption[]` | Available options |
| `layout` | `"vertical" \| "horizontal"` | Layout direction (default: `"vertical"`) |
| `haptic` | `boolean` | Haptic feedback |

### Multi select

| Prop | Type | Description |
|------|------|-------------|
| `mode` | `"multi"` | Multi-selection mode |
| `value` | `string[]` | Selected values |
| `onChange` | `(value: string[]) => void` | Change handler |
| `options` | `OptionGroupOption[]` | Available options |
| `layout` | `"vertical" \| "horizontal"` | Layout direction (default: `"vertical"`) |
| `haptic` | `boolean` | Haptic feedback |
