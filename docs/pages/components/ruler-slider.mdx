# RulerSlider

iOS-style ruler picker with Skia rendering, momentum scrolling, decay physics, and haptic feedback.

- **Tier 3** — requires Reanimated + Gesture Handler + Skia
- **npm deps:** `@shopify/react-native-skia`, `@react-native-masked-view/masked-view`, `expo-linear-gradient`, `react-native-reanimated`, `react-native-gesture-handler`
- **Registry deps:** [Theme](/theme), [Haptics](/utils/haptics)
- **Platforms:** iOS, Android

```bash [Terminal]
npx popapp add ruler-slider
```

## Usage

<PreviewVideo src="/videos/ruler-slider.mp4" />

```tsx
import { RulerSlider } from "@/components/ui/ruler-slider";

<RulerSlider
  value={height}
  onValueChange={setHeight}
  min={100}
  max={220}
  unit="cm"
/>
```

## Custom tick marks

```tsx
<RulerSlider
  value={height}
  onValueChange={setHeight}
  min={100}
  max={220}
  step={1}
  majorStep={10}
  midStep={5}
  unit="cm"
/>
```

## Decimal values

```tsx
<RulerSlider
  value={amount}
  onValueChange={setAmount}
  min={0}
  max={100}
  step={0.1}
  decimals={1}
  unit="$"
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | — | Current value |
| `onValueChange` | `(v: number) => void` | — | Change handler |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `300` | Maximum value |
| `step` | `number` | `1` | Step increment |
| `majorStep` | `number` | `10` | Major tick interval |
| `midStep` | `number` | `5` | Mid-size tick interval |
| `unit` | `string` | — | Unit label |
| `decimals` | `number` | `0` | Decimal places |
| `fontSize` | `number` | `40` | Value display font size |
| `haptic` | `boolean` | `true` | Haptic on tick crossing |

:::note
This component uses `@shopify/react-native-skia` for hardware-accelerated rendering of the ruler marks, making it performant even at high tick densities.
:::
