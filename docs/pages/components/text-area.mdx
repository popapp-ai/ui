# TextArea

Native multiline input with filled/outline variants, label, error state, and configurable height.

- **Tier 1** — no animation dependencies
- **Registry deps:** [Theme](/theme)

```bash [Terminal]
npx popapp add text-area
```

## Usage

```tsx
import { TextArea } from "@/components/ui/text-area";

<TextArea
  label="Comments"
  placeholder="Write your thoughts..."
  value={comments}
  onChangeText={setComments}
/>
```

## Variants

<PreviewImage src="/images/textarea-variants.png" alt="TextArea variants" />

```tsx
{/* Filled (default) — native background fill */}
<TextArea
  label="Feedback"
  placeholder="Tell us what you think..."
  variant="filled"
/>

{/* Outline — subtle border, no background */}
<TextArea
  label="Bug Report"
  placeholder="Describe the issue..."
  variant="outline"
/>
```

## With Error

<PreviewImage src="/images/textarea-error.png" alt="TextArea error state" />

```tsx
<TextArea
  label="Description"
  placeholder="Min 20 characters"
  error="Please provide at least 20 characters"
  variant="outline"
  value={text}
  onChangeText={setText}
/>
```

## Custom height

```tsx
<TextArea
  label="Notes"
  minHeight={200}
  maxHeight={400}
  value={notes}
  onChangeText={setNotes}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Label above the textarea |
| `error` | `string` | — | Error message below |
| `variant` | `"filled" \| "outline"` | `"filled"` | Visual style variant |
| `minHeight` | `number` | `120` | Minimum height in pixels |
| `maxHeight` | `number` | — | Maximum height |
| `disabled` | `boolean` | `false` | Disable interaction |
| `containerStyle` | `ViewStyle` | — | Outer container style |
| `inputStyle` | `TextStyle` | — | Inner input style |

Extends `RNTextInputProps` (minus `style` and `multiline` which are handled internally).
