# TextInput

Native text input with filled/outline variants, label, error state, and icon slots.

- **Tier 1** — no animation dependencies
- **Registry deps:** [Theme](/theme)

```bash [Terminal]
npx popapp add text-input
```

## Usage

```tsx
import { TextInput } from "@/components/ui/text-input";

<TextInput
  label="Email"
  placeholder="you@example.com"
  value={email}
  onChangeText={setEmail}
/>
```

## Variants

<PreviewImage src="/images/input-variants.png" alt="TextInput variants" />

```tsx
{/* Filled (default) — native iOS-style background fill */}
<TextInput
  label="Filled"
  placeholder="Native filled input..."
  variant="filled"
/>

{/* Outline — subtle border, no background */}
<TextInput
  label="Outline"
  placeholder="Outline input..."
  variant="outline"
/>
```

## States

<PreviewImage src="/images/input-states.png" alt="Input states" />

```tsx
{/* With error */}
<TextInput
  label="With Error"
  placeholder="Required"
  error="This field is required"
  value={field}
  onChangeText={setField}
/>

{/* Disabled */}
<TextInput
  label="Disabled"
  placeholder="Can't edit"
  disabled
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Label above the input |
| `error` | `string` | — | Error message below the input |
| `variant` | `"filled" \| "outline"` | `"filled"` | Visual style variant |
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"lg"` | Input height |
| `leftSection` | `ReactNode` | — | Left icon/element |
| `rightSection` | `ReactNode` | — | Right icon/element |
| `disabled` | `boolean` | `false` | Disable interaction |
| `containerStyle` | `ViewStyle` | — | Outer container style |
| `inputStyle` | `TextStyle` | — | Inner input style |

Extends `RNTextInputProps` — all standard TextInput props are supported.
